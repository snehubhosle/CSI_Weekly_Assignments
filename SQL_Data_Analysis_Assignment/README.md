# Project Overview

This project focuses on analyzing an e-commerce sales database using SQL. The objective is to understand the dataset, retrieve useful business information, and perform different types of SQL operations to answer business-related questions.

The project covers the complete workflow starting from database creation and data validation to advanced SQL concepts such as joins, aggregation, conditional logic, and transaction management. All queries were written following SQL best practices with proper formatting, meaningful aliases, comments, and optimized query structure.
