# CSI Assignment No.1

## Project Overview
This repository contains a shopping analysis assignment. It includes cleaned and combined datasets along with a Jupyter notebook for exploratory data analysis.

## Repository Structure
- `data/`
  - `cleaned_dataset.csv` - cleaned shopping dataset
  - `Combined_dataset.csv` - combined shopping dataset
- `notebook/`
  - `shopping_analysis.ipynb` - Jupyter notebook with data analysis and visualization

## How to Use
1. Open the repository in VS Code or another code editor.
2. Open `notebook/shopping_analysis.ipynb`.
3. Run the notebook cells to explore the dataset and review analysis results.

## Notes
- The notebook relies on the CSV files in `data/`.
- If you want to re-run the analysis, make sure the datasets are available in the `data/` folder.
# CSI Assignment No. 1

This repository contains the files for a data analysis assignment.

## Project structure

- `data/`
  - `cleaned_dataset.csv` — cleaned dataset used for analysis
  - `Combined_dataset.csv` — combined dataset used for analysis
- `notebook/`
  - `shopping_analysis.ipynb` — Jupyter notebook with exploratory data analysis and insights

## Usage

1. Open `notebook/shopping_analysis.ipynb` in Jupyter Notebook or JupyterLab.
2. Run the notebook cells to explore the data and review analysis results.

## Notes

- The notebook depends on standard Python data analysis libraries such as `pandas`, `numpy`, and `matplotlib`.
- Make sure the `data/` directory remains in place so the notebook can load the dataset files correctly.
