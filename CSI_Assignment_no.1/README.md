
# CSI Assignment No. 1

This repository contains the files for a data analysis assignment.

## Project structure

- `data/`
  - `cleaned_dataset.csv` — cleaned dataset used for analysis
  - `Combined_dataset.csv` — combined dataset used for analysis
- `notebook/`
  - `shopping_analysis.ipynb` — Jupyter notebook with exploratory data analysis and insights

## Usage

1. Open `notebook/shopping_analysis.ipynb` in Jupyter Notebook or JupyterLab.
2. Run the notebook cells to explore the data and review analysis results.

## Notes

- The notebook depends on standard Python data analysis libraries such as `pandas`, `numpy`, and `matplotlib`.
- Make sure the `data/` directory remains in place so the notebook can load the dataset files correctly.
